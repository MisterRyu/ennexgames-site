# Ennex Games Website

A simple static hub for the Ennex Games project ecosystem.

This first version ships a clean landing page with:

- Hero section and brand intro
- Embedded Dream Light / lofi YouTube release
- Project overview cards
- Quick links for Astra Mobile, video content, and contact
- Responsive layout with mobile nav

## Preview

### Option 1: Open locally

1. Open `index.html` in your browser.

### Option 2: Run a local server

1. Open a terminal in `C:\Dev\ennexgames-site`
2. Run a simple server, for example:
   - `python -m http.server 8000`
   - or `npx serve .`
3. Visit `http://localhost:8000`

## Deploy to GitHub Pages

1. Confirm the repo is connected to GitHub and the branch is `main`.
2. Push changes:
   - `git add .`
   - `git commit -m "Update Ennex Games hub to V2 homepage"`
   - `git push origin main`
3. In the repository settings on GitHub, enable GitHub Pages from the `main` branch.
4. Set the site folder to `/ (root)`.
5. Visit the published URL shown in GitHub Pages settings.
